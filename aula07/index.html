Aula 07 - Estudo do CSS

A sigla CSS é Cascading style Sheets é uma linguagem utilizar para estilizar e/ou diagramar elementos HTML dentro de uam página web

O CSS funciona em conjunto com o HTML, enquanto o HTML diz o que será exibido nas páginas, o CSS diz como deverá ser exibido.
